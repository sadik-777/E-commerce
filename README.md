# 🛒 Bacola - Online Grocery Shopping Center

![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)
![TailwindCSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)
![Laravel](https://img.shields.io/badge/Laravel-FF2D20?style=for-the-badge&logo=laravel&logoColor=white)


A modern, fully responsive e-commerce web application built for a seamless online grocery shopping experience.

---

## ✨ Features

- 🗂️ **Dynamic Product Catalog** — Browse products across categories like Meats, Seafood, Bakery, Beverages, and more
- 🔥 **Best Sellers & New Arrivals** — Dedicated sections for trending and recently added products
- 🔍 **Advanced Search & Filtering** — Category-based filtering with a powerful search system
- 🛍️ **Product Modal** — Quick view with image zoom, quantity selector, wishlist & compare
- 📧 **Newsletter Integration** — Subscription system for discounts and promotional updates
- 📱 **Fully Responsive** — Optimized for desktop, tablet, and mobile
- ❤️ **Wishlist & Compare** — Save and compare your favorite products
- 🧭 **Mega Menu Navigation** — Multi-level dropdown with organized shop categories

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|

| Icons | React Icons / FontAwesome |

## 🚀 Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan serve
```
